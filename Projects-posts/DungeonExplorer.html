Dungeon Explorer

What is this?

This is a small indie game project I started as a end term project for school. The plan for the game is a short RPG Like game where you explose dungeons gain gold and improve your equipment. Nothing fancy but as my first try on a proper project it can be a challenge.

Progress

Up to the end term Exam which I started this project for I was working on it for multiple hours a day. After the exam the progress slowed down but its still a hobby project I work on everynow and then.
